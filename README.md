# Attendanceproject
This project focuses on analyzing employee attendance data using Python, Pandas, and Matplotlib. The goal is to extract meaningful insights from raw attendance records and visualize key patterns related to employee presence, absence, punctuality, and work behavior. The analysis helps in identifying trends such as most present employees, frequent late comers, early leaving patterns, overtime contributors, and weekly off utilization.

Tools & Libraries Used

Python

Pandas (Data Analysis)

Matplotlib (Data Visualization
